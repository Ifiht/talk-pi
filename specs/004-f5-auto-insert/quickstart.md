# Quickstart: F5 Auto Insert

## Verify Flow
1. Start the terminal session.
2. Double-press F5 to start voice capture.
3. Speak a short phrase.
4. Finish the voice flow and confirm the transcript is inserted into the active input automatically.
5. Verify the message is still unsent and can be edited.
6. Repeat with F10 to confirm the existing behavior still works.

## Expected Behavior
- F5 double-press triggers voice flow.
- Recognized text appears in the input buffer automatically.
- Existing typed input stays intact.
- Nothing is sent automatically.
- F10 continues to work as before.
