# Quickstart: Push-to-Talk Voice Recording

## Verify Flow
1. Open app with keyboard and microphone access.
2. Press `F10` once to start recording.
3. Speak.
4. Press `F10` again to stop recording.
5. Confirm WAV file path is shown.

## Expected Behavior
- Recording begins on first `F10` press.
- Recording stops on second `F10` press.
- UI shows active recording state while recording.
- Mic errors show a clear failure state.
- Recording path is shown after stop.
