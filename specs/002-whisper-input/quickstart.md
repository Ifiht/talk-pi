# Quickstart: Whisper Text Input

## Verify Flow
1. Start push-to-talk recording and finish a short phrase.
2. Trigger recognition on second F10 press.
3. Confirm recognized text appears in active TUI input buffer.
4. Edit text if needed.
5. Send manually.

## Expected Behavior
- Recording produces a WAV file.
- WAV gets recognized with free local Whisper.
- Transcript is pasted into current TUI input.
- Existing text stays intact.
- Failure or silence shows clear message and inserts nothing junk.
