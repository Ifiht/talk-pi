# Quickstart: Mute Extension Menu

## Verify the Menu
1. Start the terminal session.
2. Open the mute menu with the extension command.
3. Confirm the current mute state is visible.
4. Select mute.
5. Trigger a spoken assistant reply and confirm no audio plays.
6. Select unmute.
7. Trigger another spoken reply and confirm audio resumes.

## Verify Active Playback Stops
1. Start a spoken reply so audio is playing.
2. Open the mute menu and choose mute.
3. Confirm the current audio stops promptly.
4. Confirm the reply text remains visible.
5. Confirm later replies stay silent until unmuted.

## Expected Behavior
- The menu clearly shows mute and unmute actions.
- Muting silences spoken output for the current session.
- Unmuting restores spoken output.
- Text replies and typing continue to work normally.
